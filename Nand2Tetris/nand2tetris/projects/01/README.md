# Project 1: Boolean Logic

The objective of this project is to build the following logic gates:
* Not
* And
* Or
* Xor
* Mux
* DMux
* Not16
* And16
* Or16
* Mux16
* Or8Way
* Mux4Way16
* Mux8Way16
* DMux4Way
* DMux8Way

More information about this project is available on https://www.nand2tetris.org/course
